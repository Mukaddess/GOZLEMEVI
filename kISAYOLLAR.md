


🔧 Visual Studio Code Kısayolları (Güncel ve Sık Kullanılanlar)

1. ALT + SHIFT + F         
→ Kodları otomatik biçimlendir (formatla)  
→ Formater automatiquement le code

2. CTRL + ALT + ↑ / ↓      
→ İmleci yukarı veya aşağı genişlet (çoklu satır imleci)  
→ Ajouter des curseurs multiples vers le haut ou le bas

3. ALT + ↑ / ↓             
→ Satırı yukarı/aşağı taşı  
→ Déplacer la ligne vers le haut ou le bas

4. CTRL + D                
→ Aynı kelimeleri seç (çoklu düzenleme)  
→ Sélectionner le mot suivant identique (édition multiple)

5. CTRL + SHIFT + K        
→ Satırı sil  
→ Supprimer la ligne

6. CTRL + ENTER            
→ Alt satıra yeni satır aç  
→ Ajouter une nouvelle ligne en dessous

7. CTRL + /                
→ Satırı yorum satırına çevir (toggle comment)  
→ Commenter/décommenter la ligne

8. SHIFT + ALT + ↓ / ↑     
→ Satırı çoğalt (duplicate line)  
→ Dupliquer la ligne

9. CTRL + B                
→ Sol kenardaki paneli (dosya gezgini) aç/kapat  
→ Afficher/masquer l'explorateur de fichiers

10. CTRL + P               
→ Hızlı dosya arama  
→ Rechercher un fichier rapidement

11. CTRL + SHIFT + P       
→ Komut paletini aç (tüm işlemler burada)  
→ Ouvrir la palette de commandes

12. CTRL + F               
→ Sayfada metin ara  
→ Rechercher dans le fichier

13. CTRL + H               
→ Sayfada metin bul ve değiştir  
→ Rechercher et remplacer

14. CTRL + TAB             
→ Açık sekmeler arasında geçiş yap  
→ Naviguer entre les fichiers ouverts

15. CTRL + K + C           
   → Seçilen satırları yorum satırı yap (çok satırlı)  
   → Commenter la sélection (plusieurs lignes)

16. CTRL + K + U           
→ Seçilen satırların yorumunu kaldır  
→ Décommenter la sélection

17. CTRL + `               
→ Terminali aç/kapat  
→ Ouvrir/fermer le terminal intégré


18. F2                     
→ Değişken/ad yeniden adlandır  
→ Renommer un élément (variable, fonction, etc.)

19. CTRL + SPACE           
→ Otomatik tamamlama menüsünü aç  
→ Afficher les suggestions (auto-complétion)

20. CTRL + SHIFT + ALT + →  
→ Bölünmüş ekran (multi-panel görünüm)  
→ Affichage fractionné (multi-fenêtre)




📘 HTML Etiketleri – Temel Kullanımlar

1. <hr>       : Bir satır çizgi çizer (yatay çizgi)
2. <br>       : Bir satır boşluk bırakır (satır atlatır)
3. <strong>   : Kalın yapar (anlamsal olarak vurgulu)
4. <b>        : Kalın yapar (görsel olarak kalınlık verir)
5. <i>        : İtalik yapar (eğik yazar)
6. <mark>     : Yazıyı sarı renkle vurgular (işaretleme efekti)
7. <u>        : Altı çizili metin
8. <em>       : Vurgulu metin (italic gibi ama anlamsal vurgu)
9. <small>    : Küçük yazı (dipnot gibi)
10. <del>     : Üzeri çizili metin (silinmiş gösterimi)
11. <ins>     : Altı çizgili yeni eklenmiş metin
12. <sub>     : Alt simge (örneğin kimya formülleri için H₂O)
13. <sup>     : Üst simge (örneğin üstlü ifadeler: x²)
14. <code>    : Kod yazısı (monospace font)
15. <pre>     : Önceden biçimlendirilmiş metin (boşluklar korunur)
16. <blockquote> : Alıntı bloğu
17. <a href=\"\"> : Bağlantı oluşturur
18. <img src=\"\"> : Görsel eklemek için
19. <ul> / <ol> / <li> : Sırasız ve sıralı liste oluşturur
20. <div>      : Genel kapsayıcı blok
21. <span>     : Satır içi küçük kapsayıcı
